Description

Let's set up our decision tree classifier. Create the DecisionTree class that includes the functions you've implemented in the previous stages: the Gini and the weighted Gini Impurity, the splitting function for one node, and the recursive splitting function. Don't forget about the __init__() method â€” you'll need it to store the tree's root and a value representing the minimum number of samples required at a leaf node; the default value is 1. You also need to implement the fit() method that will allow the model's training. It takes the dataset and calls the recursive splitting function.

When constructing the DecisionTree class, you may want to separate the methods that are meant for internal use. These methods include the Gini functions, the one-node-split function, and the recursive splitting function. fit() is not on the list since it is used to train a model. There is a convention that a leading single underscore in a Python name of a variable/function/method, e.g. _gini_impurity(), indicates to a programmer that it is for internal use only.
Objectives

In this stage, build the DecisionTree class based on the tips above.

  1. Take the input path to the file with data;
  2. Read the data from the .csv file;
  3. The Survived column is the target variable. Make sure to separate it from the dataset with features;
  4. Initialize your custom decision tree and train the model on the given dataset.

The result of the stage is a set of log messages; they were added to the recursive function in the previous stage.

If you have corrupted CSV file, please download and place it in the test directory.
